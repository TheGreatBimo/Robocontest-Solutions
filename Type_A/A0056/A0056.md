#### Translation
Sum of Single-Digit Prime Numbers

You are given a number N. Determine the minimum number of single-digit prime numbers whose sum equals N.

#### Solution
A classic dp problem;
Let dp[i] be the the answer for i
base cases:
dp[0]=-1
dp[1]=-1
dp[2]=1
dp[3]=1
dp[5]=1
dp[7]=1

we will build the dp forward,
dp[i] = min(dp[i-p]+1) over all p where p is a single digit prime (2,3,5,7)
